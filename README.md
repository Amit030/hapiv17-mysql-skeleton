# hapiv17-mysql-skeleton
Basic boilerplate for Hapi with mysql.

## Features
It includes following modules implementation.
```
1. Hapi v17
2. Mysql
3. Winston logger
4. Bcrypt
5. hapi-auth-bearer-token
6. JWT for access token generation
7. hapi-swagger
```

## Installation
```
Fork it and run npm install
```
